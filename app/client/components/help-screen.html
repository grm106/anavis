Hilfe

Tastaturkürzel Aktion
n / s / k Moduswahl: normal / Schere / Kleber
s (halten) + Klick (links) teilt einen Analyseabschnitt (Schere)
k (halten) + Klick (links) verbindet zwei Analyseabschnitte (Kleber) beim Klicken auf eine Formteilgrenze
c schneidet eine Analyse beim Anhören einer Sounddatei
strg + n erstellt eine neue Analyse
Pfeiltasten auf/ab wählt eine Analyse aus
strg + s sichert eine markierte Analyse
Leertaste startet und stoppt die Soundwiedergabe einer ausgewählten Analyse

Quickstart

  • Erstellen Sie eine neue Analyse mit strg + n.
  • Verbinden Sie eine Analyse mit einer Sounddatei, indem Sie die Sounddatei per Drag&Drop auf den Analysebalken ziehen.
  • Markieren Sie die Analyse und starten Sie den Sound mit der Leertaste.
  • Erstellen Sie beim Anhören Unterteilungen mit der c-Taste.
  • Verschieben Sie ggf. Markierungen durch "Anfassen" der Formteilgrenzen mit der Maus.
  • Benennen Sie Formteile durch Doppelklick und Texteingabe. Setzen Sie den Cursor in den nächsten Formteil mit der TAB-Taste.
  • Färben Sie benannte Formteile mit Standardfarben ein durch Klicken auf den Pinsel.
